# GIT-CHEATSHEET

by ChrBarth

  - clone via ssh:

  `git clone git@github.com:*[user]*/*[repository]*.git`

  - check for changes:

  `git status`

  - add a file for upload:

  `git add *[file]*`

  - send update:

  ```
  git commit
  git push
  ```
  
  - other way round (if the repository on github is newer than the local one):

  `git pull`

  (add and commit stay the same)
